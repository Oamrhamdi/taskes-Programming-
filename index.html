دليل التخصصات التقنية — أقسام ومطلوب لكل قسم

صفحة سريعة تعرض الأقسام الرئيسية ومهارات / معرفة تحتاجها لتتخصص فيها

1. تطوير الويب (Web Development)

أقسام داخل التخصص ومهارات مطلوبة

أقسام
  • Front-End: HTML, CSS, JavaScript, React / Vue / Angular
  • Back-End: Node.js, Python (Django/Flask), PHP, Java (Spring)
  • Full-Stack: الجمع بين الفرونت والباك، DevOps أساسي
المطلوب لتبدأ
  • إتقان HTML, CSS, JavaScript
  • فهم HTTP, APIs, Authentication
  • قواعد بيانات: MySQL, PostgreSQL, MongoDB
  • أدوات: git, npm, Webpack/Vite

2. تطوير الألعاب (Game Development)

بنايات الألعاب ثنائية وثلاثية الأبعاد، وموبايل.

أقسام
  • 2D / 3D (Unity, Unreal, Godot)
  • Mobile Games (Unity, Cocos)
  • Game Design, Level Design, Audio
المطلوب
  • لغات: C# (Unity)، C++ (Unreal)
  • محركات: Unity أو Unreal
  • مهارات نمذجة: Blender، أساسيات الرسوم والأنيميشن

3. تطوير التطبيقات (App Development)

تطبيقات موبايل وسطح مكتب وCross-platform.

أقسام
  • Android (Kotlin/Java)
  • iOS (Swift)
  • Cross-Platform (Flutter, React Native)
المطلوب
  • معرفة لغة النظام (Kotlin/Swift) أو Flutter (Dart)
  • تصميم واجهة المستخدم UI/UX
  • التعامل مع قواعد بيانات محلية وRemote (SQLite, Firebase)

4. الأمن السيبراني (Cyber Security)

حماية الأنظمة، اختبارات الاختراق، تحليل البرمجيات الخبيثة.

أقسام
  • Penetration Testing
  • Malware Analysis
  • Digital Forensics
المطلوب
  • نظام تشغيل: Linux، أدوات مثل Kali
  • شبكات: TCP/IP, DNS, HTTP
  • أدوات: Wireshark, Metasploit, Burp Suite

5. علوم الحاسب (Computer Science)

الأساس النظري: خوارزميات، نظرية الحوسبة، نظم تشغيل.

أقسام
  • خوارزميات وهياكل بيانات
  • نظم التشغيل وقواعد البيانات
  • نظرية الحوسبة والبرمجة المتقدمة
المطلوب
  • لغات: C, C++, Java, Python
  • رياضيات: جبر خطي، احتمالات، تحليل
  • فهم تعقيد الخوارزميات Big-O

6. نظم المعلومات (Information Systems)

تقاطع بين تقنية المعلومات وإدارة الأعمال.

أقسام
  • تحليل وتصميم النظم
  • إدارة قواعد البيانات وERP
  • تحليل الأعمال ومتطلبات المستخدم
المطلوب
  • SQL, Oracle, تصميم UML
  • المعرفة بإدارة مشاريع (Agile, Scrum)
  • فهم عمليات العمل وإدارة التغيير

7. الذكاء الاصطناعي (AI)

تعلم الآلة، الرؤية الحاسوبية، معالجة اللغة.

أقسام
  • Machine Learning, Deep Learning
  • Computer Vision, NLP
  • Reinforcement Learning
المطلوب
  • Python + TensorFlow / PyTorch
  • رياضيات: تفاضل، جبر خطي، إحصاء
  • مشاريع تطبيقية ومجموعات بيانات

8. علم البيانات (Data Science)

تحليل، هندسة، وتصوير البيانات.

أقسام
  • Data Analysis, Data Engineering
  • Data Visualization, Big Data
المطلوب
  • Python/R, Pandas, NumPy
  • SQL, Hadoop, Spark
  • مهارات عرض البيانات: Matplotlib, Plotly

9. هندسة الحاسب (Computer Engineering)

عتاد وبرمجيات مدمجة، FPGA، دوائر.

أقسام
  • تصميم العتاد (Digital/Analog)
  • Embedded Systems, Microcontrollers
  • FPGA، VHDL/Verilog
المطلوب
  • لغات: C, Assembly
  • إلكترونيات، دوائر رقمية
  • معرفة FPGA وARM

10. أمن الشبكات (Network Security)

حماية البنية التحتية للشبكة والسياسات الأمنية.

أقسام
  • Network Design, Firewalls
  • IDS/IPS, VPNs
المطلوب
  • CCNA / CCNP مفيد
  • فهم البروتوكولات: TCP/IP, BGP
  • معدات Cisco وLinux

11. أمن المعلومات (Information Security)

سياسات، تشفير، إدارة مخاطر والتوافق.

أقسام
  • Data Protection, Identity & Access Management
  • Risk Management, Compliance (GDPR, ISO 27001)
المطلوب
  • معرفة التشفير Cryptography
  • سياسات الأمن وإدارة الحوادث
  • شهادات مثل CISSP مفيدة
